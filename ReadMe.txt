*********************ReadMe**************************************

This project builds a network in mininet.
Five files are submitted in this assigment.

mytopo.py: 
1. create a topology 
2. set ARP table on each host 
3. add floodlight controller to the net
4. start the net in CLI

flowrules.py:
1. create flow table for each host
2. insert above table to each host via REST API

newhost.sh:
add a new host to the network

newrules.py:
add new flow table for the added host

ReadMe.txt:
instruction for this project
